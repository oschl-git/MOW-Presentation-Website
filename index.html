MOW / a website for practicing knowledge

MOW allows you to create custom sets of questions and answers, which are then used to test your knowledge. It can be a great tool when studying for exams or remembering anything that's important to you.


Many ways to practice / there are 4 practice modes in total, including modes that allow you to swap questions and answers.

Always synced / learning sets are hosted on a server, accessible from anywhere.

No limitations / create as many learning sets as you want with as many questions as you need.


This web app was made as a final project for my third year of studying at SPŠE Ječná. Although technically functional, it isn't stable or ready to be deployed in production, and therefore isn't accessible in a working environment. If you are interested in seeing it function, you can find the source code on GitHub and do whatever you please with it.


Used tools / NodeJS, Express, EJS, VSCode, GIMP
